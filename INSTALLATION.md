# Installation Instructions

This document provides setup instructions for developers to get started with the SAEB hub.

## Prerequisites
- [Node.js](https://nodejs.org/) (version x.x.x)
- [Git](https://git-scm.com/)

## Setup Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/joandersonsilva7845-lang/SAEB-hub.git
   cd SAEB-hub
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```

## Running Tests
To run tests, use:
```bash
npm test
```
